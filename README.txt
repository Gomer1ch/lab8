Завдання 1:
За допомогою ВСІХ ПЕРЕРАХОВАНИХ графічних примітивів (лінії, 
прямокутники, кола, сплайни і криві Безьє) створіть малюнок об'єкта (у 
кожного студента повинен бути свій об'єкт, без повторень!)).
Запрограмувати наступні дії:
1. Зміна розміру об'єкта в заданий число раз.
2. Переміщення / зміни положення об'єкта в заданому напрямку по 
натисканні на кнопки (вгору / вниз / вліво / вправо).
3. Розмістити n об'єктів по заданій траєкторії. Траєкторія задається 
графіком (пряма, крива.)
4. Розмістити n об'єктів по заданій траєкторії і зі зменшенням / 
збільшенням розміру.
